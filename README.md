# Snoopy 快樂算一算

國小一年級加減法練習小遊戲，單一 HTML 檔案，無需後端，手機也能玩。

## 特色

- Snoopy 經典狗屋造型與 Peanuts 朋友角色互動
- 點擊骨頭數數、連續答對獎勵、爵士風背景音樂
- 加法與減法兩種模式，每輪 10 題

## 開始玩

直接用瀏覽器開啟 [`kids-math-game.html`](kids-math-game.html) 即可。

本地預覽：

```bash
python3 -m http.server 8765
# 開啟 http://127.0.0.1:8765/kids-math-game.html
```

## 線上遊玩

GitHub Pages 部署後可於以下網址遊玩（部署完成後更新）：

https://YOUR_USERNAME.github.io/kids-math-game/kids-math-game.html
